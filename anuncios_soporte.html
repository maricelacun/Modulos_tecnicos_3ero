Componentes de un Sistema

Interna

Los componentes internos incluyen:

  • Case
  • Mainboard
  • Fuente de poder
  • Procesador
  • Memorias
  • Disco duro
  • Tarjetas de expansión
  • Unidades de almacenamiento
  • Buses de datos
  • Setup
  • BIOS
  • Pila
  • Ensamblaje y conexión

Externa

Los componentes externos incluyen:

  • Dispositivos periféricos:
    • Monitor
    • Teclado
    • Mouse
    • Scanner
    • Impresora
    • Parlantes y micrófonos
    • Proyectores
  • Unidades de almacenamiento externo:
    • Pendrive
    • CD
    • DVD
    • Tarjetas de memoria
    • Discos duros externos
